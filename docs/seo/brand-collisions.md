# Brand Collision Audit Report

This document records the search brand collision audit performed for the name **EmberChamber**.

## Audited Search Queries

1. `"EmberChamber"`
2. `"EmberChamber messenger"`
3. `"EmberChamber app"`

---

## Identified External References & Collisions

### 1. `ember-messenger` (Ember.js Addon)
- **Result URL:** https://emberobserver.com/addons/ember-messenger
- **Why it may confuse users:** Developers or users searching for "Ember messenger" or "Ember messaging app" may land on this npm/Ember addon repository rather than the EmberChamber project website.
- **Official/Unofficial:** Unofficial (independent project).
- **Recommended Action:**
  - **Create official social profiles/listings:** Establish clear, descriptive GitHub repository tags and official docs explaining that EmberChamber is a standalone secure messenger, not related to the Ember.js ecosystem.
  - **Ignore:** The addon is specific to Ember.js frontend notifications and does not overlap in function.

### 2. *True Dragon of Valiria* (Web Novel Location)
- **Result URL:** Various webnovel sites.
- **Why it may confuse users:** Searches for "EmberChamber" without messaging context will return chapters of this fiction work where the "Emberchamber" is a vault.
- **Official/Unofficial:** Unofficial.
- **Recommended Action:**
  - **Ignore:** Creative fiction references present no business or domain naming threats and will naturally rank lower as EmberChamber messaging search visibility grows.

### 3. Magical Cave Generator (TTRPG Cavern Naming)
- **Result URL:** https://www.scabard.com
- **Why it may confuse users:** "Ember Chamber" is a randomly generated cave name in fantasy roleplaying tools.
- **Official/Unofficial:** Unofficial.
- **Recommended Action:**
  - **Ignore:** No product confusion is possible.

### 4. Gmail Account Namespace (`emberchamber@gmail.com` and fallback)
- **Result URL:** N/A (Google Sign-up)
- **Why it may confuse users:** The username `emberchamber` is already taken on Gmail (third-party registration). Google suggested alternatives like `emberchamber515@gmail.com` or `chamberember237@gmail.com`.
- **Project Resolution:** The fallback address **`emberchamber.com@gmail.com`** has been registered and is designated as the official project-linked Google account.
- **Recommended Action:**
  - **Use Custom Domain Email:** Route official external correspondence through custom domain emails (e.g., `@emberchamber.com`).
  - **Use for Consoles:** Use `emberchamber.com@gmail.com` only when an explicit Google/Gmail account is required (e.g., Google Play Console, Google Analytics).

---

## Strategic Recommendations

- **Secure Key Handles:** Secure official social media and developer platform profiles (e.g. Twitter/X handles, LinkedIn organization page, Google Play Developer console name) under the exact brand string `EmberChamber` to prevent trademark squatting.
- **Anchor Text Discipline:** When linking to the homepage, use consistent anchor text such as "EmberChamber Encrypted Messenger" rather than generic words.
